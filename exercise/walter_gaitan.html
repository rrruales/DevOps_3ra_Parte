Examen Git y Github

Alumno: Walter Gaitan

  1. Que es un sistema de control de versiones?

    Es una herramienta que funciona para llevar un historial ordenado de todos los cambios que se hacen para un proyecto o repositorio. Gracias a estas herramientas se puede salvar el estado de un proyecto y agregar detalles sobre los cambios realizados, tambien se pueden recuperar estados anteriores en caso de realizar un cambio no deseado.

  2. Cual es la diferencia entre git y github?

    Git es un sistema de control de versiones, este funciona mayormente de manera local. Github es un repositorio en la nube donde subir tus proyectos.
    Por ejemplo: al colaborar en un proyecto, puedes copiarlo a tu dispositivo y llevar un control de tus cambios con Git y, una vez listos los cambios, puedes compartir los cambios en el repositorio en linea para que el resto de tu equipo pueda verlo.

  3. Cómo creo una rama en git?

    Una vez tengas el repositorio copiado en tu ordenador, puedes utilizar la terminal para ejecutar el siguiente comando git checkout -b nombre-de-nueva-rama

  4. Cómo clono un repositorio?

    1. Una vez en el repositorio a clonar, presionar el boton clonar
    2. Si se tiene el dispositivo agregado por SSH en github, seleccionar SSH, de no ser asi, utilizar HTTPS
    3. Copiar la direccion dada
    4. En tu dispositivo, abrir terminal y utilizar el siguiente comando git clone urlDeRepositorio.git
